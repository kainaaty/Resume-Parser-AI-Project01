RESUME PARSER AI PROJECT

Project Description:
This project is an AI-based Resume Parser developed using Python, NLTK, and Pandas. The system reads resumes from a dataset, cleans the text using NLP techniques, detects important skills, assigns scores to resumes, and ranks the top candidates automatically.

Features:

* Resume text cleaning
* Skill detection
* Resume scoring
* Candidate ranking
* CSV output generation

Technologies Used:

* Python
* Pandas
* NLTK
* Scikit-learn

Project Workflow:

1. Load resume dataset
2. Clean resume text
3. Detect skills
4. Assign resume scores
5. Rank top candidates
6. Save final output

Output:
The project generates a Final_Resume_Output.csv file containing detected skills and resume scores.
